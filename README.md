## How to run the project:
1. run npm run dev, npm run preview or npm run build command then type the given local address on your browser.
2. When opened you are in the Home page. There's a header where you can see the login button, courses and tutors link.
3. When pressed on login, there will be a modal form which gets your user email and password.
5. After the login process, you go back to home page. Your login button will have your user name instead and now the button has a drop down menu with 2 options, allowing you to logout or go to dashboard.
6. If you are not signed in yet, there will also be a link in the login modal form and also a button in the homepage which you can sign from there.
7. When on register page there are some fields for you to complete. You can also choose your role as a tutor or as a student.
